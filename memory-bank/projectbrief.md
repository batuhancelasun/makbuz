# Project Brief: Makbuz Expense Tracker

## Overview
Makbuz is a self-hosted, PWA-enabled expense tracking application with AI-powered receipt scanning capabilities. Built with Node.js backend and vanilla HTML/CSS/JavaScript frontend, designed for personal/home lab use.

## Core Purpose
Provide a simple, fast, and beautiful expense tracking solution that allows users to:
- Track expenses and income manually
- Scan receipts using AI (Google Gemini Flash) to auto-fill expense data
- View spending analytics with visual charts
- Track individual items purchased across receipts
- Set up recurring expenses/income
- Access the app offline (PWA support)

## Key Requirements
1. **Self-hosted**: Deploy on user's own server
2. **PWA**: Installable, works offline
3. **Receipt Scanning**: AI extracts place, date, amount, and items from photos
4. **Simple Storage**: JSON file-based (no database required)
5. **Docker Ready**: Easy deployment via Docker Hub
6. **Modern UI**: Clean, minimal design with dark/light mode
7. **Item Tracking**: Track and analyze most frequently purchased items
8. **Income Support**: Track both expenses and income
9. **Recurring Transactions**: Support for recurring expenses/income

## Target Users
- Home lab enthusiasts
- Individuals wanting simple expense tracking
- Users who want self-hosted solutions
- People who want to track spending without complex budgeting features

## Success Criteria
- Fast and simple to use
- Beautiful, modern UI
- Reliable receipt scanning
- Easy deployment and setup
- Works offline
- Tracks items for analysis
